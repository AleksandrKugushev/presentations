Упомянуть:

- Рассказать Почаевцу
- Подумать о том, что Почевец говорил, когда увлекаешься

Сделать:
- Вспомнить что говорили на тренинге
- Test Last: добавить картинку + лучше объяснить. Возможно, придумать физический аналог.
- Test First: добавть картинку + СИЛЬНО ЛУЧШЕ ОБЪЯСНИТЬ. Привести аналогию из реальной жизни или показать пример разработки кода.
- TDD: картинка + акцентировать на отличие от Test Last. Четко показать, что главное тут - итеративность. Возможно, отметить TLD, но только если не будет перегруза. Возможно, сделать иконку Red Green Refactoring
- Слайд-План, где еще раз проходимся по каждому из видов.
- Более развернутый пример Test First подхода с Elasticsearch. Показать проблему - сложный API и хрен его воспроизведешь.
- Еще раз прогнать все, и отметить слайды, гед иедт зависание
- На слайде про легаси заменить плохой код на какашки
- Каждый раз, когда упоминаю Code Retreat - сделать слайд + объяснить что это такое.
